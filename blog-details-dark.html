The Four Fundamentals of Color Grading

C olor grading is a complicated subject that one may find it hard to comprehend in one sitting. Therefore I've simplified it into four fundamentals so that even beginners can find it easy to understand.

So, the four fundamentals are:
1. Brightness (lightness)
2. Contrast
3. Color Balance (Tone)
4. Saturation

10 th March
By : Dennis

SOUNDS EASY ENOUGH RIGHT?This is always how I begin my first lesson. It sounds simple enough, and the basic control for these four fundamentals are already easily accessible from the get-go, so everyone can start grading right away.

However, you can't master the art of color grading using simple and imprecise tools. You must be meticulous every step of the way. Therefore every professional colorist should learn every tools available to them and know which tool to choose to get the best results. But the fundamentals always stay the same, I still approach every images in these four directions.
